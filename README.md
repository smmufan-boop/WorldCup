# 足球红黑榜网页

入口文件：

`G:\weixin\reports\red_black_board\index.html`

功能：

- 页面分为“今日方案”和“既往红黑榜”两个标签页，点击标签切换。
- 今日方案使用固定模板：足球分析师视角、最终推荐方案、赔率模型推测、自有模型校准、风险控制。
- 今日方案展示未赛推测，赛后再回填到既往红黑榜。
- 按日期分组。
- 按系列赛分组。
- 红球表示命中，黑球表示失误；页面标签显示为“红”“黑”。
- 绿球表示待赛/未结算方案。
- 黄球表示半中/混合，蓝球表示整场主状态走水。
- 卡片标签会额外标出“半中”“走水”；例如某场整体是红，但其中 DNB 推盘，也会在卡片上显示“走水”。
- 点击既往足球卡片可以查看赛前方案、结算、复盘和经验校准；点击今日方案卡片可以查看固定分析模板。
- 顶部支持关键词、日期、系列赛、结果筛选；“只看走水”会匹配结算中含“推/走水”的比赛，“只看半中/混合”会匹配结算中含“半赢/赢半/半输”的比赛。
- 折算命中率按“一场只算一次”计算，并取该场最优结算结果：
  - 全赢/命中/胜 = 1
  - 半赢/半中 = 0.5
  - 走水/推 = 0.5
  - 黑 = 0

维护方式：

- 既往结果数据内嵌在 `index.html` 的 `records` 数组里。
- 今日方案数据内嵌在 `index.html` 的 `todayPlans` 数组里。
- 后续有新赛果，把对应 `todayPlans` 方案转入 `records`，字段包括：
  - `date`
  - `series`
  - `match`
  - `score`
  - `status`: `hit` / `miss` / `mixed` / `push`
  - `summary`
  - `plan`
  - `settlement`
  - `review`
  - `lessons`
